# Archive Notice

This repository has been **archived** and is no longer actively maintained.

## What happened?

`oh-my-customcode` was a framework for managing Claude Code and OpenAI Codex agent systems. The project has been deprecated in favor of a **self-contained, Claude-native only** approach.

## Why?

- The dual-provider (Claude + Codex) architecture added unnecessary complexity
- Consumer projects like [baekgom-agents](https://github.com/baekenough/baekgom-agents) have evolved to be fully self-contained
- Claude Code's built-in features (hooks, skills, auto memory) have matured enough to eliminate the need for an external framework

## What should I use instead?

If you were using `oh-my-customcode`, transition to managing your `.claude/` directory directly. Claude Code's native features now cover what this framework provided.

## Historical Reference

The last stable release was **v0.11.0**. All code remains available for reference in this archived state.
