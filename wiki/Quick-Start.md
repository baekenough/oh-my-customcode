# Quick Start

## Installation

```bash
# Install globally
npm install -g oh-my-customcode

# Initialize in your project
cd your-project
omcustom init
```

That's it. You now have a fully configured Claude Code environment.

## Language Options

```bash
# English (default)
omcustom init

# Korean
omcustom init --lang ko
```
