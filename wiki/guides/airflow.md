# Airflow Guide

Reference documentation for Apache Airflow DAG development and workflow orchestration best practices.

## Overview

Apache Airflow is a platform for programmatically authoring, scheduling, and monitoring workflows. This guide covers DAG authoring patterns, task design, scheduling, connections management, monitoring, and performance optimization. Used primarily by the `de-airflow-expert` agent for data engineering pipeline work.

## Key Topics

- DAG authoring patterns and TaskFlow API
- Task design and dependency management
- Testing Airflow DAGs
- Scheduling strategies and SLAs
- Connections and Variables management
- Monitoring and alerting
- Performance optimization for large DAGs
