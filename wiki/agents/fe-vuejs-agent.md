# fe-vuejs-agent

Expert Vue.js (Vue 3) developer for Composition API, single-file components, Pinia state management, Vue Router navigation, and Vite build tooling.

## Overview

`fe-vuejs-agent` specializes in Vue 3's Composition API and `<script setup>` syntax, reactive state with ref/reactive/computed, single-file components with scoped CSS, Pinia for state management, and Vue Router for navigation. It also applies `impeccable-design` and `web-design-guidelines` skills to ensure visual quality and accessibility compliance.
