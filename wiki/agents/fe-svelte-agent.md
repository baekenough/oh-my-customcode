# fe-svelte-agent

Expert Svelte developer for compiler-based reactivity, reactive statements, Svelte stores, and SvelteKit full-stack development with SSR.

## Overview

`fe-svelte-agent` specializes in Svelte's unique compiler-based reactivity model — `$:` reactive statements, two-way bindings, actions, and lifecycle hooks — as well as SvelteKit for full-stack development with file-based routing, server-side rendering, and load functions. It applies `impeccable-design` and `web-design-guidelines` skills for visual quality and accessibility compliance.

## Key Details

- **Model**: sonnet
- **Domain**: frontend
- **Tools**: Read, Write, Edit, Grep, Glob, Bash
- **Skills**: `impeccable-design`, `web-design-guidelines`
- **Memory**: project
- **Effort**: medium
