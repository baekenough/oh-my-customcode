# be-nestjs-expert

Expert NestJS developer for opinionated, scalable Node.js applications with TypeScript, enterprise API architectures, and decorator-based patterns.

## Overview

`be-nestjs-expert` builds enterprise-grade NestJS applications using TypeScript's full type system. It implements the full NestJS patterns: `@Injectable`/`@Module`/`@Controller` decorators, dependency injection, Pipe/Guard/Interceptor middleware layers, DTO validation with class-validator/class-transformer, and authentication/authorization guards (JWT, roles). Targets teams needing Angular-style structure for backend Node.js services.
