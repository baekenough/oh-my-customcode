# Kotlin Best Practices

Idiomatic Kotlin patterns from JetBrains style guide and effective Kotlin.

## Overview

Reference patterns for idiomatic Kotlin: null safety (safe calls, elvis operator, non-null assertions), data classes, sealed classes, extension functions, coroutines and Flow, scope functions (let, run, with, apply, also), companion objects, and smart casts. Covers interop with Java and Kotlin-specific Spring Boot patterns. Used by `lang-kotlin-expert`.
