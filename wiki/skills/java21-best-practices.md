# Java 21 Best Practices

Modern Java 21 patterns from Virtual Threads, Records, and Sealed Classes.

## Overview

Reference patterns for Java 21: Virtual Threads (Project Loom) for high-concurrency I/O, Records for immutable data carriers, Sealed Classes for exhaustive pattern matching, text blocks, switch expressions, and pattern matching for instanceof. Covers migration from older Java versions and JVM tuning for virtual thread workloads. Used by `lang-java21-expert`.
