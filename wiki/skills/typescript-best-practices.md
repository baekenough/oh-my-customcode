# TypeScript Best Practices

Type-safe TypeScript patterns from the TypeScript Handbook and community standards.

## Overview

Reference patterns for TypeScript: strict mode configuration, discriminated unions, type narrowing, utility types (Partial, Required, Pick, Omit, ReturnType), generics best practices, `const` assertions, template literal types, module augmentation, branded types for domain modeling, and `zod` for runtime validation.
