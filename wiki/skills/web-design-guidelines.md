# Web Design Guidelines

UI code review with 100+ rules for production-quality web interfaces.

## Overview

Comprehensive ruleset for web UI code review covering 100+ criteria: accessibility (WCAG 2.1 AA), responsive design (mobile-first), CSS architecture (BEM, CSS custom properties), performance (Core Web Vitals), color contrast, typography scale, touch target sizes, animation performance (GPU-accelerated), dark mode support, and internationalization. Used by `fe-design-expert` during UI code review.
