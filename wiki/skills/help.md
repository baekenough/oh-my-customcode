# Help

Show help information for commands, agents, and skills.

## Overview

Displays contextual help for the oh-my-customcode system. Shows available slash commands with their argument hints and descriptions. Provides quick-start examples and links to relevant guides. Can be invoked with a specific command name to show detailed help for that command.

## Key Details

- **Scope**: harness
- **User-invocable**: yes
- **Command**: `/omcustom:help`
- **Effort**: not specified
