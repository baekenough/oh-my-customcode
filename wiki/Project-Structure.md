# Project Structure

After `omcustom init`:

```
your-project/
├── CLAUDE.md              # Entry point for Claude
├── .claude/
│   ├── rules/             # Behavior rules
│   ├── hooks/             # Event hooks
│   ├── contexts/          # Context files
│   ├── agents/            # Agent definitions (flat .md files)
│   └── skills/            # Skill definitions
└── guides/                # Reference docs
```
